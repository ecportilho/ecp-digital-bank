Produção real

Todas as transações financeiras passam pelo ECP Pay. Os bancos PF e PJ utilizam a API para transferências via Pix e pagamentos com cartão. O FoodFlow integra pagamentos de delivery diretamente pelo gateway, com split automático para os restaurantes.
